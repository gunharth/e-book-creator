Possible Markup for Text editing

[t]Title[et] 									// bold green text
[b]bold[eb]										// bold text
[i]bold[ei]										// italic text
[l=http://www.e-book-creator.at]Home[el]			// "Home" links to http://www.e-book-creator.at

[email=support@e-book-creator.at]email[eemail]	// "email" becomes mailto link to support@e-book-creator.at

[img]img/fine.jpg[eimg]							// adds an image relative to the document

[anchor]anchortext[eanchor]						// adds a target for a link within a page
To link to the above anchor on that same page: [l=#anchortext]my anchor link[el]

[buttonSignUp]Sign Up![ebuttonSignUp]			// creates the sign up button eg.: on Homepage
